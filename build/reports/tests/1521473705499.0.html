We added more network profiles

Wanted to check your service with high packet losses? Maybe server latency? You now can easily do that using the new profiles weve added. Check it out!

Firefox now in Private Beta

While we are working hard to improve the testRTC backend and service, theres also a lot of work that goes on to improving the availability of our service on as many devices as possible.

To that end, weve been hard at work in recent months to get testRTC to work with Firefox as well. We are at a point when the implementation is rather stable. While we do know of some issues we have with Firefox, wed like to start collecting customer feedback about it if you want to try it out, then do contact us about our private beta for Firefox.
